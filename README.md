**Terminal Space** is an AI-powered news summarizer that pulls stories from multiple sources and presents a clear, bias-aware snapshot of what's going on in the world. It cuts through clutter and condenses information using large language models to give you a fast, no-BS understanding of current events.

---

## Features

- Aggregates news from multiple APIs
- Summarizes and compares articles with GPT-4o
- Simple, clean interface built with Replit Agent
- Responsive design for mobile and desktop
- Zero-fluff, factual headlines in seconds
